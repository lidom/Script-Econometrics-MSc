Econometrics Data Sets

Welcome to the Econometrics Data Page! This folder contains the 22 data sets used in Econometrics for illustrations, applications, and all end-of-chapter exercises. Each dataset can be found in one of the folders. The names of the folders correspond to the names of the datasets given in the textbook.

In each folder you will find four files. One is a Data Description file. I strongly encourage you to read this file, as it describes the meaning of the individual variables. The other files are alternative-format versions of the data set. For each data set, you will find a Stata-formatted, Excel-formatted, and a text-formatted version. The same variables are found in each format. The choice of data-formatting is a matter of preference and convenience.

The datasets as presented here are (in many cases) processed and/or abridged versions of the original research datasets. The original source datasets and code which created the processed datasets is posted in the file Econometrics Source Data on the author's textbook webpage.

In addition, a complete achieve of the R, Stata, and Matlab code used for the applications, illustrations, and figures in Econometrics is posted in the file Econometrics Programs on the author's textbook webpage.

For more information about the author, see his webpage.
